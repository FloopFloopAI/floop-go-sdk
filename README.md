# floop-go-sdk

Official Go SDK for the [FloopFloop](https://www.floopfloop.com) API — **coming soon**.

Until this SDK ships, use one of:

- **Node.js**: [`@floopfloop/sdk`](https://github.com/FloopFloopAI/floop-node-sdk)
- **Python**: [`floopfloop`](https://github.com/FloopFloopAI/floop-python-sdk)
- **CLI**: [`floop`](https://github.com/FloopFloopAI/floop-cli) — every command supports `--json` for scripting from Go via `os/exec`
- **Raw HTTP**: the API is a simple Bearer-token REST surface at `https://www.floopfloop.com/api/v1/*` — see the other SDKs for the route shapes

Track progress / open feature requests at
[`github.com/FloopFloopAI/floop-go-sdk/issues`](https://github.com/FloopFloopAI/floop-go-sdk/issues).
